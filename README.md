# Mythical World Tree

Welcome to the **Mythical World Tree** project! This application is designed to immerse users in a fantastical realm where they can explore the majestic world tree, its branches, roots, and the various realms it connects. 

## Features

- **World Tree Visualization**: Experience the mythical world tree with interactive components representing its branches, roots, and canopy.
- **Realms Exploration**: Journey through different realms including the Sky Realm, Midgard Realm, and Underworld Realm, each with unique visuals and interactions.
- **Spirits and Creatures**: Encounter various mythical creatures and spirits that inhabit the world tree, each with their own stories and interactions.
- **Journal Entries**: Keep track of your journey by creating journal entries and inscribing magical runes.
- **Cosmic Animations**: Enjoy stunning animations that enhance the mystical experience throughout the application.

## Project Structure

The project is organized into several key directories:

- **src/components**: Contains all the React components for the world tree, realms, spirits, journal, and UI elements.
- **src/pages**: Includes the main pages for the application, integrating various components.
- **src/hooks**: Custom hooks for managing logic related to the world tree, mythical journeys, realm transitions, and spirit bonding.
- **src/utils**: Utility functions for transformations, calculations, and animations.
- **src/constants**: Constants defining realm configurations, mythical stages, and spirit types.
- **src/styles**: CSS styles for mythical themes, realm gradients, and cosmic animations.
- **src/assets**: SVG assets for tree shapes and mystical symbols.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd mythical-world-tree
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and visit `http://localhost:3000` to see the application in action!
